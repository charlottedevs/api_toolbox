# ApiToolbox

Handy methods to interact with Charlotte Junior Devs [API](https://github.com/charlottejuniordevs/api)

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'api_toolbox'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install api_toolbox
